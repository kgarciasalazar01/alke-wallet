# Alke Wallet
Aplicación web para gestionar envíos de dinero, depósitos y transacciones de manera simple y clara.

---
Posee las siguientes funcionalidades:
- **Index** con Titulo, credenciales y autor.
- **Login de usuario** para acceder de forma segura.
- **Enviar dinero** a otros usuarios.
- **Depositar fondos** en tu cuenta.
- **Historial de transacciones** para revisar movimientos.
- **Menú principal** con navegación intuitiva.

---

## Tecnologías utilizadas
- **HTML5** para la estructura.
- **CSS** para el diseño y estilos.
- **JavaScript** para la lógica y la interacción.
